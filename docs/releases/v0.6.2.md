# Release Notes - v0.6.2

**Release Date:** February 2026
**Type:** Patch release

---

## Summary

Version 0.6.2 packages the latest streaming/runtime fixes, Databricks system test hardening, and repository workflow updates into a single patch release.

This release also includes updated Databricks workspace IaC assets and deployment/release documentation alignment.

---

## Highlights

- Streaming execution and recovery updates for multi-input orchestration and restart behavior.
- Databricks system test flow improvements, including config override handling via test harness conventions.
- Added Databricks workspace diff tooling and Terraform workspace configuration scaffolding.
- Documentation refresh across setup, build, release, and developer workflow guides.
- Continued alignment of release/deploy tasks with Poetry + Poe commands.

- `kindling_synapse-0.6.2-py3-none-any.whl`

## Upgrade Guide

### To 0.6.2

1. Build platform wheels:
   ```bash
   poetry run poe build
   ```
2. Deploy wheel(s) to ADLS:
   ```bash
   poetry run poe deploy --platform fabric
   poetry run poe deploy --platform synapse
   poetry run poe deploy --platform databricks
   ```
3. Re-run relevant system suites:
   ```bash
   poetry run poe test-system --platform databricks
   poetry run poe test-system --platform synapse
   poetry run poe test-system --platform fabric
   ```
