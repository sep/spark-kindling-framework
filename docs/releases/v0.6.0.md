# Release Notes - v0.6.0

**Release Date:** February 2026
**Type:** Minor release

---

## Summary

Version 0.6.0 is the current stable framework baseline for multi-platform data apps across Fabric, Synapse, and Databricks.

This release aligns build, deployment, and documentation workflows around the unified Poetry + Poe task model.

---

## Highlights

- Unified deploy workflow with `poe deploy --platform ...` and `poe deploy --release ...`
- Current framework version baseline set to `0.6.0`
- Platform-specific wheel distribution through GitHub Releases
- Updated documentation for release operations and environment configuration

---

## Release Assets

Expected wheel assets attached to the GitHub release:

- `kindling_fabric-0.6.0-py3-none-any.whl`
- `kindling_synapse-0.6.0-py3-none-any.whl`
- `kindling_databricks-0.6.0-py3-none-any.whl`

---

## Upgrade Guide

### From 0.5.x to 0.6.0

1. Update framework version and rebuild wheels:
   ```bash
   poetry run poe version --bump_type patch
   poetry run poe build
   ```
2. Deploy the platform wheel(s) you need:
   ```bash
   poetry run poe deploy --platform fabric
   poetry run poe deploy --platform synapse
   poetry run poe deploy --platform databricks
   ```
3. Validate bootstrap and configuration loading in your target workspace.

- Platforms: Microsoft Fabric, Azure Synapse Analytics, Databricks
